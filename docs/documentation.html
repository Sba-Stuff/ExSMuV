ExSMuV Setup Documentation

Getting Started

  1. Make sure you have x64 bit Windows Operating System.
  2. Create a folder named ExSMuV anywhere on your PC.
  3. Create three subfolders: chromedriver, chrome, and server inside ExSMuV.
  4. Download Chromedriver and extract it into the chromedriver folder.
  5. Download Chrome and extract it into the chrome folder.
  6. Note: Close any running Chrome browser (from system tray) to avoid conflicts.
  7. Download a Server (e.g., WAMP, XAMPP, UniServer) and extract it into the server folder.
  8. Note: Make sure the curl library is enabled on the server.
  9. Open the chromedriver folder, type cmd in the address bar, and run: 
    chromedriver.exe --port=49400
  10. Open the chrome folder and run Chrome.exe.
  11. Download the GitHub code "ExSMuV" and place it inside the htdocs or www directory of the server.
  12. Start the preferred server (e.g., UniServer).
  13. Open localhost in the Chrome browser you launched earlier.
  14. Try out the “ExSMuV” Software.
  15. Sample searches for “Chemistry” and “Harry Potter” are cached for demo.
  16. When done, close the browser, command prompt, and server.
